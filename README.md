# claude-remote

Monitor and control Claude Code sessions from your phone.

## Quickstart

```bash
# Install deps
make setup

# Start dev server at http://localhost:8000
make run
```

## Development

```bash
# Run tests
make test

# Lint
make lint

# Format in place
make format

# Type-check
make typecheck

# Full check gate (lint + typecheck + test)
make check
```

## Requirements

- Python 3.12
- [uv](https://docs.astral.sh/uv/) — `pip install uv` or see uv docs
